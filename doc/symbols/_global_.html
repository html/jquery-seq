eachStep(collection, callback, endcallback)
Map array collection to deferred callback. Callback executes on every element after previous execution was finished.
Defined in: jquery-seq.js. 
eachStep(["Hello world", "Goodbye world"], function(hello){
   var def = $.Deferred();

   setTimeout(function(){
     alert(hello);
     def.resolve();
   }, 1000);

   return def;
}, function(){alert("Just said good bye");});
Parameters:
{Array} collection
A collection of items to map one after another
{Function} callback
A callback that will be called on every item from collection. Callback should return deferred object.
{Function} endcallback Optional
A callback that will be called after every collection element is mapped.

sequentialExecute(collection, endcallback)
Executes array of deferred callbacks one after another.
Defined in: jquery-seq.js. 
sequentialExecute([
  function(){ 
    var def = $.Deferred()

    setTimeout(function(){
      alert("Hello world");
      def.resolve();
    }, 1000);

    return def;
  },
  function(){ 
    var def = $.Deferred()

    setTimeout(function(){
      alert("Goodbye world");
      def.resolve();
    }, 1000);

    return def;
  }], function(){ alert('All executed'); });
